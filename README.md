# Now TV Toolkit

A toolkit built in React for Now TV

Install for development
---

To install `nowtv-toolkit` locally for development:

```bash
git clone git@github.com:rogerfranklin2/nowtv-toolkit.git
cd nowtv-toolkit
npm install
```

Then run `webpack` to compile
